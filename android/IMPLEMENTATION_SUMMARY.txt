🚀 使用方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 克隆仓库
   git clone https://github.com/xeonliu/SJTU-Canvas-Helper.git

2. 打开项目
   cd SJTU-Canvas-Helper/android
   用 Android Studio 打开

3. 同步 Gradle
   等待 Gradle 同步完成

4. 运行应用
   连接设备或启动模拟器
   点击运行按钮 ▶️

5. 登录使用
   输入 Canvas Token
   开始使用
